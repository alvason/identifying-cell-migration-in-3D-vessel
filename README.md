## Installation
### Prerequisites
This code is written and tested in Python 3.6.5
The required Python libaries are:
* NumPy
* SciPy
* Matplotlib
* AlvaMachinery

### Getting started
```
clone git https://github.com/alvason/identifying_neurite_by_RRS.git
```
